Création du Backend Django

1-/Créer un environnement virtuel
python -m venv venv

Activùtion :
venv\Scripts\activate

2-/Installation des dépendances

pip install django djangorestframework djangorestframework-simplejwt

3-/Création du projet
django-admin startproject expense_tracker
cd expense_tracker

4-/Création de l'application API
python manage.py startapp api

Maintenant la configuration de l'application

5-/Modifier settings.py (expense_tracker/settings.py)

Ajoute dans INSTALLED_APPS :
INSTALLED_APPS = [
    ...
    'rest_framework',
    'api',
]

Ajouter config JWT
Toujours dans settings.py, ajoute à la fin :

from datetime import timedelta

REST_FRAMEWORK = {
    'DEFAULT_AUTHENTICATION_CLASSES': (
        'rest_framework_simplejwt.authentication.JWTAuthentication',
    ),
}

SIMPLE_JWT = {
    'ACCESS_TOKEN_LIFETIME': timedelta(minutes=60),
}

7-/Appliquer les migrations
python manage.py migrate

8-/Et enfin Lancer le serveur
python manage.py runserver

lien : `http://127.0.0.1:8000`

# 🔑 Étape 1 — Obtenir le token JWT

Ton URL pour récupérer le token est :

```
http://127.0.0.1:8000/api/token/
```

### Méthode :

* Méthode : **POST**
* Body : JSON

```json
{
  "username": "ton_nom_admin",
  "password": "ton_mot_de_passe_admin"
}
```

Réponse attendue :

```json
{
  "refresh": "…",
  "access": "…"
}
```

* `access` = token que tu vas utiliser pour t’authentifier
* `refresh` = pour renouveler l’access quand il expire

---

# 🔑 Étape 2 — Utiliser le token

Dans Postman ou ton frontend mobile :

* Header :

```
Authorization: Bearer <ton_access_token>
```

* URL API exemples :

```
GET http://127.0.0.1:8000/api/categories/
POST http://127.0.0.1:8000/api/expenses/
```

POST body pour créer une dépense :

```json
{
  "category": 1,
  "amount": "25.50",
  "description": "Repas midi"
}
```

Angular / React Native pourra utiliser exactement la même structure.
